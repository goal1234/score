

模型思考：
    基于逻辑斯蒂回归模型，自然增长曲线的变化（出生数量）开始很少，然后繁衍增长迅速最后几乎不怎么增长来的。
    其增长最快的是在概率0.5附近，应为其导数在附近发生了方向上的变化所以其形成了一个分类方法。手动的特征选择，
    好的数据优于好的模型。对缺失值没有好的办法。

撸模型的姿势:
	1．确定分类变量, 足够自然或者能够反映的自然，手动找这个曲线贴合...
	2．异常数据处理 -> 剔除，填充，插值等进入重复步骤 ->
	3．计算KS, IV，WOE的值
		- 分类的标签在其他数据上分的好不好， 标签区别其他变量的强度
		- 强度还有权重，最后是用权重的LR，标签区别好的变量有更高的权重

	4．建立模型，变量选取AUC,p,bias
	5．在区间内出现（0,1）出现频率的情况进行切割               
	6．交叉验证，时间内外
	7．寻找点进行woe归一化					   #证据权重转换，给数据哪部分什么权重，对应好的权重大，在后面模型中应该会直接
	8．将归一化的结果进行回归
	9．求取odds和评分
	10．得到评分卡片                                           #每个特征多少分+基础分，，做了个模型给特征打分
	11．模型校准
