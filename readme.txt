

模型思考：
    基于逻辑斯蒂回归模型，自然增长曲线的变化（出生数量）开始很少，然后繁衍增长迅速最后几乎不怎么增长来的。
    其增长最快的是在概率0.5附近，应为其导数在附近发生了方向上的变化所以其形成了一个分类方法。手动的特征选择
    仍然是模型的决定因素，好的数据（指向力强的而非质量）优于好的模型。同时算法对缺失值没有好的办法。可以这样
    的想一想：目标变量的0:1和每个变量的0:1:2:3...这样的对应关系形成了自然增长曲线，所以样本的0:1比例是有关系的
    一般是1:1，联想下繁衍，也可以改一下，不过这似乎不符合原始的公式毕竟这是系数估计模型。。。

撸模型的姿势:
	1．确定分类变量, 足够自然或者能够反映的自然，手动找这个曲线贴合...
	2．异常数据处理 -> 剔除，填充，插值等进入重复步骤 ->
	3．计算KS, IV，WOE的值
		- 分类的标签在其他数据上分的好不好， 标签区别其他变量的强度
		- 强度还有权重，最后是用权重的LR，标签区别好的变量有更高的权重

	4．建立模型，变量选取AUC,p,bias
	5．在区间内出现（0,1）出现频率的情况进行切割               #想一想自然增长曲线
	6．交叉验证，时间内外
	7．寻找点进行woe归一化					   #证据权重转换，给数据哪部分什么权重，对应好的权重大，在后面模型中应该会直接
	8．将归一化的结果进行回归
	9．求取odds和评分
	10．得到评分卡片                                           #每个特征多少分+基础分，，做了个模型给特征打分
	11．模型校准

输出的什么？
   概率值变成了成组的分，归集概率成了一个组，每个特征的分来直接反映，受到其他影响的时候数据变换那么模型就需要更新
   简而言之，为了区分一组顺序（很多东东里挑想要的），将一组序列{x1...xn}按照自然增长曲线的方式进行转换(WOE和LR模型确定)
   输出了评分（概率归集）。
